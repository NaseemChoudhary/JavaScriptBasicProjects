# ThemeSwithcer

A theme switcher demo that toggles between light and dark modes.

![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript&logoColor=black) ![Status](https://img.shields.io/badge/Status-Completed-success)

---

## 📖 Overview

This project is built using standard web technologies and focuses on interactive DOM behavior, responsive styling, and clean UI design. It is intended as a learning project for practicing JavaScript event handling, CSS layout, and browser compatibility.

## ✨ Features

* Built with **HTML5**, **CSS3**, and **Vanilla JavaScript**.
* Clean and responsive layout for desktop and mobile screens.
* User-friendly interactive controls and animations.
* Easy local setup with no build tools required.

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)

## 🚀 Usage

1. Open the `index.html` file in a modern web browser.
2. Use the controls or inputs on the page to interact with the app.
3. Observe updates and results directly in the browser.

## 📁 Project Structure

## 💻 Installation

No installation is required for this static project. Simply open `index.html` in your preferred browser.

## 📌 Notes
