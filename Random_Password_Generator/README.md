## Overview

**Password Generator** is a lightweight front-end application that helps users generate secure passwords based on their own preferences.

It’s designed for anyone who wants:

* stronger passwords,
* more control over character types,
* and a fast, no-friction experience directly in the browser.

There’s nothing to install, no accounts required, and no data ever leaves the page.

---

## Features

* Custom password length (up to 50 characters)
* Toggle character sets:

  * Lowercase letters
  * Uppercase letters
  * Numbers
  * Symbols
* One-click password generation
* One-click copy to clipboard
* Clean, modern glassmorphism UI
* Fully client-side (no backend, no tracking)

## Usage

1. Enter a desired password length
2. Select at least one character option
3. Click **Generate Password**
4. Click **Copy** to save it to your clipboard

The password is generated instantly and never stored.
